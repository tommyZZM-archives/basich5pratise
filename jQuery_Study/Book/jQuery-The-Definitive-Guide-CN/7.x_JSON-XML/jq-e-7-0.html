JavaScript操作JSON和XML

创建一只JSON字符串的步骤

  1. var obj = {}; //创建一个对象( 等价于var obj = new Object() )
  2. var obj.a = "tommy"; //给对象创建a属性并赋值
    (你也可以在创建对象的时候创建属性 var obj = {'a':"tommy"})
  3. var json = JSON.stringify(obj); //把obj对象转换成为json字符串
