jQuery操作数组中的元素

使用$.each()遍历数组中的元素或对象属性

jQuery.each(object, [callback])

如果能用for循环遍历，则尽量使用for循环

使用$.grep()筛选数组中的元素

jQuery.grep(array, callback, [invert])
  • array:待过滤数组。
  • callback:此函数将处理数组每个元素。第一个参数为当前元素，第二个参数而元素索引值。此函数应返回一个布尔值。另外，此函数可设置为一个字符串，当设置为字符串时，将视为“lambda-form”（缩写形式？），其中 a 代表数组元素，i 代表元素索引值。如“a > 0”代表“function(a){ return a > 0; }”。
  • invert:为 false 或为设置，则函数返回数组中由过滤函数返回 true 的元素，当"invert" 为 true，则返回过滤函数中返回 false 的元素集。

使用$.map()变更数组中的元素或对象

jQuery.map(arr|obj,callback)
  • arr|obj:待过滤数组|对象。
  • callback:为每个数组元素调用，而且会给这个转换函数传递一个表示被转换的元素作为参数。 函数可返回任何值。另外，此函数可设置为一个字符串，当设置为字符串时，将视为“lambda-form”（缩写形式？）， 其中 a 代表数组元素。如“a * a”代表“function(a){ return a * a; }”

使用$.inArray()搜索元素的位置

jQuery.inArray(value,array,[fromIndex])结果返回的是元素的引索值

功能与indexOf()类似

  • value:用于在数组中查找是否存在
  • array:待处理数组。
  • fromIndex:用来搜索数组起始位置，默认值为0。
